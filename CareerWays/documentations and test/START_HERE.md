## ⚡ The Fastest Way to Get Started

### Windows
```powershell
pip install -r requirements.txt
python backend/init_db.py
cd backend
python app.py
# Then open: frontend/index.html in your browser
```

### Mac/Linux
```bash
pip3 install -r requirements.txt
python3 backend/init_db.py
cd backend
python3 app.py
# Then open: frontend/index.html in your browser
```

1. **Students answer** one simple question about themselves
2. **AI analyzes** their response using NLP and machine learning
3. **System recommends** 22 college courses based on their profile
4. **Students explore** course details and save favorites
